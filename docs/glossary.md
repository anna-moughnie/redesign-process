# Glossary

## Hero Image

Banner image for website that is usually found at the top of its homepage.

## User Experience (UX)

A user's interactions with and feelings about a product.

## User Interface (UI)

Tools and visual elements provided to a user to interact with a product.

## Wireframe

Sketch of an app of a website's user interface without content, colours, or images as to focus on drafting page layout.
